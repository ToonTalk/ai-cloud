Snap! is a big brother of Scratch. It can do much more than Scratch but is a bit more complex. It is documented here.

Most importantly it has enabled the eCraft2Learn project to add new blocks. These include
  1. the ability to use speech input and output
  2. image recognition by connecting to AI cloud services (free API keys required)
  3. the ability to read and write the digital pins of a Raspberry Pi or Arduino connected to a Raspberry Pi. Note that this requires the running of a local server on the Raspberry Pi. This has already been performed if you are using an eCraft2Learn SD card on a Raspberry Pi.
These blocks are available here and documented here.

For controlling Arduinos beyond simple digital sensors and actuators we recommend Snap4Arduino which adds many new blocks for programming Arduinos.
