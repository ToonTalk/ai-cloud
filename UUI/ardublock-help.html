ArduBlock is a block-based Arduino programming language. Unlike Snap! and Snap4Arduino it is not web-based but runs as a Java application. The main unique feature of this programming language is that programs can be compiled to run directly on an Ardunio. This means the Ardunio can be embedded in an artefact and run without either being accompanied by a Raspberry Pi or in communication with another computer.

The eCraft2Learn project provides no support or integration for this tool. Download instructions are here.
